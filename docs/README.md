# PhotoGeoView Documentation

このディレクトリには、PhotoGeoViewプロジェクトに関するすべてのドキュメントが含まれています。

## 📁 ディレクトリ構造

```
docs/
├── summaries/           # 開発サマリー・統合レポート
├── guides/              # ユーザーガイド・開発ガイド
├── reports/             # 分析レポート・セキュリティレポート
├── specifications/      # プロジェクト仕様書
└── ai_integration/      # AI統合ドキュメント
```
