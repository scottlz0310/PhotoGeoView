# Photo Geo View 機能改善追加案 2025/11/21

- V2.0.2をリリースしたところであるが、ユーザーフィードバックによりさらなる改善点が明らかになった。
- 以下に、今後のバージョン(v2.1.x)で検討すべき機能改善案を列挙する。

## 1. UI関係

- Exif情報パネルは情報の粒度を絞りステータスバーに表示してパネルには表示させなくする
- 表示項目は後述する表示メニューで設定可能にする

### 1.1 メニューバーの追加

- ファイル、表示、設定、ヘルプ
- 常時表示にするか、三本線メニューアイコンをクリックで展開にするかを比較検討する

### 1.1.1 ファイルメニューの追加

- フォルダを開く、終了（メインウィンドウのSelect Folderボタンを廃止）

### 1.1.2 表示メニューの追加

- メインウィンドウに配置したボタン類の機能を集約させてメインウィンドウをシンプルにする
- テーマ切替、パネルオンオフ
- ステータスバーに移動させたExif各情報の表示/非表示選択
- ドロップダウンリスト＋チェック形式で実装

### 1.1.3 設定メニューの追加

- パネル配置のカスタマイズ機能を追加
  - 構成要素はフォルダナビゲータ＋ファイル内容、サムネイルパネル、地図パネル、画像表示エリアの4つ
  - 上記要素を縦2横2のグリッドに配置できるようにする
- 関連付けの設定メニューの追加
  - 画像ファイル形式ごとに関連付けを設定できるようにする
  - インストーラーでの関連付け設定オプションも追加

### 1.1.4 ヘルプメニューの追加

- アプリケーションの使い方ガイド
- ショートカットキー一覧
- バージョン情報メニューの追加
- 更新機能の呼び出しメニューの追加
- 多言語設定(当面は日本語、英語)

## 2. 起動方法の多様化

- 画像ダブルクリックでの起動
- 画像ドラッグアンドドロップでの起動
- 画像の右クリックメニューからの起動
- すべて、ファイルが存在しているフォルダを開くようにし、当該ファイルを選択状態にする
