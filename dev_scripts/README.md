# Development Scripts

このディレクトリには開発・検証用のスクリプトが含まれています。
本番環境では使用されません。

## スクリプト分類

### デバッグスクリプト (debug_*)
- デバッグ・問題調査用の一時的なスクリプト

### テストスクリプト (test_*)
- 統合テストや検証用の実験的スクリプト
- 正式なテストは `tests/` ディレクトリにあります

### 修正スクリプト (fix_*)
- 特定の問題修正のための一時的なスクリプト

### デモスクリプト (demo_*, simple_*)
- 機能デモ・動作確認用のシンプルなスクリプト

### その他
- `analyze_workflow.py` - ワークフロー分析
- `create_test_file.py` - テストファイル生成
- `minimal_test.py` - 最小限の動作テスト
- `run_ci_safe.py` - CI セーフモード実行
- `start_production.py` - 本番環境起動スクリプト
- `verify_theme_menu.py` - テーマメニュー検証

## 注意事項

これらのスクリプトは開発過程で作成された検証用のもので、
コードの品質や保守性は保証されていません。
参考用として保管されていますが、適宜削除・整理されます。
